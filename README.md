# RAZOR — MkDocs Material

Este repositorio contiene el código fuente del blog **RAZOR** usando **Material for MkDocs**.

## Cambios incluidos en este ZIP

- Paleta visual en **tonos morados oscuros** (con **modo claro** y **modo oscuro**).
- Logo en navbar **más grande** y sin solaparse con el título.
- Botones con **nuevo estilo** (pill + glow sutil + hover).
- Cards de artículos con:
  - **Imagen** (entre tags y descripción)
  - Mejor layout (grid) y sombras modernas
  - Animaciones suaves al aparecer y al hacer hover
- Buscador con tipografía más compacta ("Búsqueda" más pequeña).

## Deploy

Solo haz push a `main`.

GitHub Actions desplegará automáticamente en GitHub Pages con `mkdocs gh-deploy`.
